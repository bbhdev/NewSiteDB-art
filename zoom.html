Calibrated decoration

A line should sit just under each heading.

Browser zoom (Ctrl/Cmd + / -) scales text physically, but a fixed-viewport SVG layer sized in vw / vh stays the same physical size on screen. So decoration that fits the text at 100% drifts at any other zoom.

This page calibrates a few shapes against the real text rects at page load, then locks those viewBox coords in place. From there you can zoom the page and toggle the counter-scale fix on/off to see what the runtime change would look like.

Watch the magenta marks

The marks under each heading and around this phrase are positioned via viewBox coordinates that don't change. With the fix ON, the SVG layer scales with zoom and they should track the text. With it OFF, they should drift.
